# Магазин одежды

Учебный проект интернет-магазина одежды на Python с использованием NiceGUI.

## Возможности

- Просмотр каталога товаров
- Фильтрация по категории и размеру
- Поиск по названию
- Корзина покупок
- Адаптивный интерфейс

##  Технологии

*   **Язык:** Python 3.8+
*   **NiceGUI:**  — для создания богатого веб-интерфейса на чистом Python.
*   **База данных:** SQLite — легковесная встроенная база данных для хранения товаров.

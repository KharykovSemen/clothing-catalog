#  Каталог одежды

Учебный проект интернет-магазина одежды на Python с использованием NiceGUI и SQLite.

Функции:
- Просмотр каталога товаров
- Поиск по названию
- Добавление новых товаров
- Редактирование существующих товаров
- Цветные карточки товаров для наглядности

Технологии:

- Python 3.9+
- NiceGUI - для создания веб-интерфейса
- SQLite - для хранения данных

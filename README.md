# Магазин одежды 

Веб-приложение для интернет-магазина одежды с удобным каталогом, системой фильтрации и корзиной покупок. Разработано на Python с использованием фреймворка NiceGUI.

## 🎯 Описание проекта

Простое и функциональное веб-приложение для магазина одежды, которое позволяет пользователям просматривать каталог товаров, фильтровать их по различным параметрам, добавлять в корзину и оформлять заказы. Приложение имеет чистый и интуитивно понятный интерфейс.

## ✨ Основные возможности

- **Каталог товаров** с отображением в виде сетки карточек
- **Поиск товаров** по названию с подсветкой совпадений
- **Фильтрация** по категории (Мужская/Женская) и размеру (XS-XL)
- **Корзина покупок** с возможностью добавления и удаления товаров
- **Автоматический подсчет** общей стоимости
- **Система учета остатков** на складе
- **Адаптивный дизайн** для комфортного просмотра на разных устройствах

## 🛠 Технологии

- **Python 3.8+** - основной язык разработки
- **NiceGUI** - фреймворк для создания веб-интерфейса
- **SQLite3** - легковесная база данных для хранения товаров
- **Dataclasses** - для структурированного представления данных

## 🚀 Установка и запуск

### Шаги по установке
## 1.Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или
venv\Scripts\activate  # для Windows
## 2.Установите зависимости
pip install -r requirements.txt
## 3.Запустите приложение
python app.py
## 4.Откройте браузер и перейдите по адресу:
http://localhost:8080
